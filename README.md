## Components :point_up:
- Arduino Uno x1.
- Ultrassonic sensor x1.
- Servo motor x1.
- Hobby whells x2.
- Motor Shield L293D x1.
- Motor DC 3-6V with reduction box x2.
- 9V Duracell Battery x1.
- 1,5V Rayovac AA Battery x4.
- Cardboard body.
- Jumpers xN.
